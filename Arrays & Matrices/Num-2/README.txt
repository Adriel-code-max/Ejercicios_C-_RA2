Suma de Dos Arreglos en C#
Este programa en C# permite al usuario ingresar dos arreglos de 10 números cada uno, calcula la suma de todos los elementos de ambos arreglos y muestra el resultado final.

Características
Entrada de datos: El usuario ingresa 10 números para cada arreglo.

Procesamiento: Suma todos los elementos de ambos arreglos.

Salida: Muestra el resultado total de la suma.

Ejecución
El programa solicita al usuario que ingrese 10 números para el primer arreglo.

Luego, solicita 10 números para el segundo arreglo.

Calcula la suma de todos los elementos de ambos arreglos.

Muestra el resultado de la suma en la consola.

Tecnologías
Lenguaje: C#

Entorno: .NET
