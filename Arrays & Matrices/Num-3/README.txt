Gestión de Calificaciones de Estudiantes en C#
Este programa en C# permite al usuario ingresar los nombres y calificaciones de una cantidad específica de estudiantes. Luego, muestra la lista de estudiantes junto con sus respectivas calificaciones.

Características
Entrada de datos: El usuario ingresa el número de estudiantes, sus nombres y calificaciones.

Procesamiento: Almacena los nombres y calificaciones en arreglos.

Salida: Muestra los nombres de los estudiantes y sus calificaciones.

Ejecución
El programa solicita al usuario que ingrese la cantidad de estudiantes.

Luego, pide el nombre y la calificación de cada estudiante.

Finalmente, muestra la lista de estudiantes con sus respectivas calificaciones.

Tecnologías
Lenguaje: C#

Entorno: .NET
