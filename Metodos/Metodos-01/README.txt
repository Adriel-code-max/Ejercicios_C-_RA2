Programa de Métodos en C#
Este programa en C# implementa varios métodos para realizar operaciones básicas, como sumar dos números, multiplicar dos números, verificar si un número es par y convertir una cantidad de dólares a pesos (utilizando una tasa de cambio fija).

Características
Métodos implementados:

Suma de dos números: Recibe dos números enteros y devuelve su suma.

Multiplicación de dos números: Recibe dos números enteros y devuelve su producto.

Verificación de número par: Recibe un número entero y verifica si es par o no.

Conversión de dólares a pesos: Recibe una cantidad en dólares y la convierte a pesos utilizando una tasa de cambio fija (1 dólar = 60 pesos).

Entrada de datos:

Números ingresados por el usuario para cada operación.

Salida:

Muestra los resultados de las operaciones en la consola.

Ejecución
El programa solicita al usuario que ingrese dos números para sumar y muestra el resultado.

Solicita dos números para multiplicar y muestra el resultado.

Solicita un número para verificar si es par y muestra el resultado.

Solicita una cantidad en dólares para convertir a pesos y muestra el resultado.

Tecnologías
Lenguaje: C#

Entorno: .NET
