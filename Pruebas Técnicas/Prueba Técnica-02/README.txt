Sistema de Registro Académico en C#
Este programa en C# simula un sistema de registro académico que permite gestionar estudiantes y sus calificaciones en diferentes asignaturas. Los usuarios pueden agregar estudiantes, asignar calificaciones y mostrar los registros académicos.

Características
Gestión de estudiantes:

Permite agregar estudiantes con su nombre y edad.

Gestión de asignaturas:

Permite agregar asignaturas con cuatro calificaciones por asignatura.

Calcula el promedio de las calificaciones.

Funcionalidades:

Agregar estudiante: Permite ingresar datos de estudiantes y asignaturas.

Mostrar registros: Muestra la información de los estudiantes y sus promedios en las asignaturas.

Menú interactivo: Permite al usuario elegir entre las opciones disponibles y volver al menú principal.

Ejecución
El programa inicia mostrando un menú con las opciones disponibles.

El usuario puede:

Agregar estudiantes y asignaturas.

Mostrar los registros académicos.

Después de cada operación, el programa pregunta si el usuario desea volver al menú principal o salir.

Tecnologías
Lenguaje: C#

Entorno: .NET
