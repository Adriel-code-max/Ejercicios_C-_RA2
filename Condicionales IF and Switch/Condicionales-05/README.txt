Verificador de Números Positivos y Negativos en C#
Este programa en C# permite al usuario ingresar un número entero y determina si es positivo o negativo.

Características
Entrada de datos:

Un número entero.

Procesamiento:

Verifica si el número es mayor o igual a cero (positivo) o menor a cero (negativo).

Salida: Muestra un mensaje indicando si el número es positivo o negativo.

Ejecución
El programa solicita al usuario que ingrese un número entero.

Verifica si el número es positivo (mayor o igual a 0) o negativo (menor a 0).

Muestra un mensaje indicando si el número es positivo o negativo.

Tecnologías
Lenguaje: C#

Entorno: .NET
