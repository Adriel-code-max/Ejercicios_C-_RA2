Calculadora de Promedio y Aprobación en C#
Este programa en C# permite al usuario ingresar cuatro notas de un estudiante, calcula su promedio y determina si el estudiante aprobó o reprobó basándose en un criterio de aprobación (nota >= 70).

Características
Entrada de datos:

Cuatro notas del estudiante.

Procesamiento:

Calcula el promedio de las cuatro notas.

Verifica si el promedio es mayor o igual a 70 para determinar si el estudiante aprobó o reprobó.

Salida: Muestra el promedio y un mensaje indicando si el estudiante aprobó o reprobó.

Ejecución
El programa solicita al usuario que ingrese las cuatro notas del estudiante.

Calcula el promedio de las notas.

Si el promedio es mayor o igual a 70, muestra un mensaje de aprobación.

Si el promedio es menor a 70, muestra un mensaje de reprobación.

Tecnologías
Lenguaje: C#

Entorno: .NET
