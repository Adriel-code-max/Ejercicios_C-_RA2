Comparador de Tres Números en C#
Este programa en C# permite al usuario ingresar tres números enteros y determina cuál es el mayor de ellos.

Características
Entrada de datos:

Tres números enteros.

Procesamiento:

Compara los tres números para determinar cuál es el mayor.

Salida: Muestra un mensaje indicando el número mayor.

Ejecución
El programa solicita al usuario que ingrese tres números enteros.

Compara los tres números para determinar cuál es el mayor.

Muestra un mensaje indicando el número mayor.

Tecnologías
Lenguaje: C#

Entorno: .NETT
