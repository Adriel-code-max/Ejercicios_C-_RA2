Ordenador de Tres Números en C#
Este programa en C# permite al usuario ingresar tres números enteros y los ordena de mayor a menor, mostrando el número mayor, el central y el menor.

Características
Entrada de datos:

Tres números enteros.

Procesamiento:

Compara los tres números para determinar cuál es el mayor, el central y el menor.

Salida: Muestra los números ordenados de mayor a menor.

Ejecución
El programa solicita al usuario que ingrese tres números enteros.

Compara los números para determinar cuál es el mayor, el central y el menor.

Muestra los números ordenados de mayor a menor.

Tecnologías
Lenguaje: C#

Entorno: .NET
