Verificador de Números Pares e Impares en C#
Este programa en C# permite al usuario ingresar un número entero y determina si es par o impar.

Características
Entrada de datos:

Un número entero.

Procesamiento:

Verifica si el número es divisible entre 2 (par) o no (impar).

Salida: Muestra un mensaje indicando si el número es par o impar.

Ejecución
El programa solicita al usuario que ingrese un número entero.

Verifica si el número es par (divisible entre 2) o impar (no divisible entre 2).

Muestra un mensaje indicando si el número es par o impar.

Tecnologías
Lenguaje: C#

Entorno: .NET
