Comparador de Números en C#
Este programa en C# permite al usuario ingresar dos números enteros y determina cuál es el mayor o si ambos son iguales.

Características
Entrada de datos:

Dos números enteros.

Procesamiento:

Compara los dos números para determinar cuál es mayor o si son iguales.

Salida: Muestra un mensaje indicando el número mayor o si ambos son iguales.

Ejecución
El programa solicita al usuario que ingrese dos números enteros.

Compara los dos números para determinar cuál es mayor o si son iguales.

Muestra un mensaje indicando el número mayor o si ambos son iguales.

Tecnologías
Lenguaje: C#

Entorno: .NET
