Calculadora de Descuentos en C#
Este programa en C# calcula el descuento aplicado a un artículo basándose en su precio. Dependiendo del rango de precio, se aplica un porcentaje de descuento específico. Si el precio no entra en ningún rango, no se aplica descuento.

Características
Entrada de datos:

Precio del artículo.

Procesamiento:

Verifica el rango de precio y aplica el descuento correspondiente:

Precio >= 1000: 30% de descuento.

Precio >= 5000: 50% de descuento.

Precio >= 10000: 80% de descuento.

Precio entre 15000 y 20000: 10% de descuento.

Si el precio no entra en ningún rango, no se aplica descuento.

Salida: Muestra el precio original y el precio con descuento (si aplica).

Ejecución
El programa solicita al usuario que ingrese el precio del artículo.

Verifica el rango de precio y aplica el descuento correspondiente.

Muestra el precio original y el precio con descuento.

Si no aplica descuento, muestra un mensaje indicando que no hay descuento.

Tecnologías
Lenguaje: C#

Entorno: .NET
