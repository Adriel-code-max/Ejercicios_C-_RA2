Calculadora Básica en C#
Este programa en C# es una calculadora simple que permite al usuario realizar operaciones básicas (suma, resta, multiplicación y división) con dos valores ingresados.

Características
Operaciones soportadas:

Suma

Resta

Multiplicación

División

Entrada de datos: El usuario ingresa dos valores numéricos.

Salida: Muestra los resultados de las cuatro operaciones.

Ejecución
El programa solicita al usuario que ingrese dos valores numéricos.

Realiza las operaciones de suma, resta, multiplicación y división con los valores ingresados.

Muestra los resultados de cada operación en la consola.

Tecnologías
Lenguaje: C#

Entorno: .NET
