Generador de Patrón Triangular en C#
Este programa en C# permite al usuario generar un patrón triangular de asteriscos (*) basado en el número de filas que ingrese.

Características
Entrada de datos:

Número de filas para el patrón triangular.

Procesamiento:

Utiliza dos bucles for anidados:

El bucle externo controla el número de filas.

El bucle interno imprime los asteriscos (*) en cada fila.

Salida:

Muestra un patrón triangular de asteriscos en la consola.

Ejecución
El programa solicita al usuario que ingrese el número de filas para el patrón triangular.

Utiliza un bucle externo para iterar sobre cada fila.

Dentro de cada fila, utiliza un bucle interno para imprimir los asteriscos correspondientes.

Muestra el patrón triangular en la consola.

Tecnologías
Lenguaje: C#

Entorno: .NET
