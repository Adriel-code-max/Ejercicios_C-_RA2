Sumatoria de Números Ingresados en C#
Este programa en C# permite al usuario ingresar números de manera continua y calcula la sumatoria de todos los números ingresados. El programa termina cuando el usuario ingresa el número 0.

Características
Entrada de datos:

Números ingresados por el usuario de manera continua.

Procesamiento:

Utiliza un bucle while para solicitar números hasta que se ingrese 0.

Calcula la sumatoria de todos los números ingresados.

Salida:

Muestra la sumatoria total de los números ingresados.

Ejecución
El programa solicita al usuario que ingrese números de manera continua.

Si el usuario ingresa 0, el programa termina.

Calcula la sumatoria de todos los números ingresados.

Muestra la sumatoria total en la consola.

Tecnologías
Lenguaje: C#

Entorno: .NET
