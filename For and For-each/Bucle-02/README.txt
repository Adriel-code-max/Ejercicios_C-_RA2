Verificador de Contraseña en C#
Este programa en C# permite al usuario ingresar una contraseña y verifica si es correcta. El usuario tiene un máximo de 3 intentos para ingresar la contraseña correcta.

Características
Entrada de datos:

Contraseña ingresada por el usuario.

Procesamiento:

Compara la contraseña ingresada con una contraseña predefinida.

Permite un máximo de 3 intentos.

Salida:

Muestra un mensaje indicando si la contraseña es correcta o incorrecta.

Si se agotan los intentos, muestra un mensaje indicando que se ha excedido el número de intentos permitidos.

Ejecución
El programa solicita al usuario que ingrese la contraseña.

Compara la contraseña ingresada con la contraseña predefinida.

Si la contraseña es correcta, muestra un mensaje de éxito y termina el programa.

Si la contraseña es incorrecta, permite hasta 3 intentos.

Si se agotan los intentos, muestra un mensaje indicando que se ha excedido el número de intentos permitidos.

Tecnologías
Lenguaje: C#

Entorno: .NET
